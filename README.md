Q1.What is JSX, and why is it used?
ANS:JSX is a syntax extension for JavaScript (JS) that looks like HTML and it is used with React. It allows developers to write UI components in a more readable and declarative way by mixing markup with JS logic.

Q2.What is the difference between State and Props?
ANS:Props are used o pass data from a parent component to child component and read-only. 
State is managed within a component itself and can change over time, usually in response to user actions

Q3.What is the useState hook, and how does it work?
ANS:It's a react function that lets you add state to functional components. It works by returning an array with two values.

Q4.How can you share state between components in React?
ANS:By lifting the states up to a common parent component and passing it down as props. 

Q5.How is event handling done in React?
ANS:It is the process of managing user interactions using event handler functions. It uses camelCase syntax and passes functions instead of strings unlike plain HTML
